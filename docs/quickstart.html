Quickstart

Installing Inkycal

Install inkycal directly via the GitHub repo:

# clone the repo
git clone https://github.com/aceinnolab/Inkycal

# go to downloaded repo folder
cd Inkycal

# install Inkycal
pip3 install -e ./
